# Category Icons v2

Modern, minimalist line-art icons for menu categories with active/inactive states.

## Available Icons

### Active State (Red)
- `all-active.png` - Grid icon for showing all categories
- `starters-active.png` - Bowl with spoon icon
- `mains-active.png` - Chicken leg icon
- `sides-active.png` - French fries icon
- `desserts-active.png` - Cupcake icon
- `drinks-active.png` - Beverage glass with straw
- `hookah-active.png` - Hookah/shisha icon

### Inactive State (Gray/Dark)
- `all-inactive.png`
- `starters-inactive.png`
- `mains-inactive.png`
- `sides-inactive.png`
- `desserts-inactive.png`
- `drinks-inactive.png`
- `hookah-inactive.png`

## Design Style

- Clean, minimalist line art
- Active: Red color (#FF0000 or similar)
- Inactive: Gray/dark color for unselected state
- Consistent stroke width
- Simple, recognizable shapes
- Works well on both light and dark backgrounds

## Migration from v1

The v1 icons used PNG files with active/inactive states. The v2 icons maintain the same naming pattern but with a more modern, minimalist design aesthetic.
