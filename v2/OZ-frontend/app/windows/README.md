# OrderZap Windows Desktop App

Electron + React desktop application for menu management.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

This will start:
- Vite dev server on http://localhost:5173
- Electron app window

## Build

```bash
npm run build
npm run build:electron
```

## Features

- View all menu items
- Add new menu items
- Edit existing items
- Toggle stock status
- Add images via URL
- Real-time sync with Convex backend
